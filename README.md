# Travel Recommendation Website

A responsive travel recommendation website built with HTML, CSS, and JavaScript that helps users discover ideal destinations based on their preferences.

## Features

- **Home Page**: Hero section, introduction, and categorized recommendations
- **About Us Page**: Company information, team members, and values
- **Contact Us Page**: Contact form and company contact information
- **Navigation Bar**: Responsive navigation with search functionality
- **Search Functionality**: Search destinations by name, type, or description
- **Responsive Design**: Works on all device sizes
